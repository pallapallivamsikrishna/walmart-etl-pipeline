# Walmart Retail Data ETL Pipeline

## Overview
This project implements an end-to-end ETL pipeline using Python and SQL on Walmart retail sales data.

## Objectives
- Extract retail dataset from Kaggle
- Clean and transform data using Pandas
- Load processed data into SQL database
- Prepare structured data for analysis

## Tools Used
- Python
- Pandas
- SQL
- Kaggle
- Jupyter Notebook

## Project Files
- walmart_etl_pipeline.ipynb
- walmart_etl_pipeline.py
